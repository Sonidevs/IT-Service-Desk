# ICT Service Help Desk System

## Description
The ICT Service Help Desk System is a web-based application designed to help users report ICT issues and allow technicians to track, manage, and resolve support tickets efficiently.

## Features
- User ticket submission
- Technician ticket management
- Issue tracking and status updates
- Admin dashboard
- Secure login system

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP
